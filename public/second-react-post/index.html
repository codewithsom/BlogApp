This post aims to cover advanced React.js concepts that can help you elevate your React skills and enhance your application’s performance and maintainability.

1. React Context API

Overview

The Context API provides a way to share values between components without having to pass props down manually through every level of the component tree. It is particularly useful for managing global state, theming, or authentication across your application.

Key Concepts

  • Context Creation: Use React.createContext to create a context object.
  • Provider: The Provider component allows you to pass the context value down the component tree.
  • Consumer: The Consumer component allows you to access the context value in any component that is a descendant of the Provider.

Example

// Creating Context
const ThemeContext = React.createContext('light');

// Creating Provider Component
const ThemeProvider = ({ children }) => {
  const [theme, setTheme] = React.useState('light');
  
  return (
    <ThemeContext.Provider value={{ theme, setTheme }}>
      {children}
    </ThemeContext.Provider>
  );
};

// Consuming Context
const ThemedButton = () => {
  const { theme, setTheme } = React.useContext(ThemeContext);
  
  return (
    <button
      style={{ background: theme === 'light' ? '#fff' : '#333', color: theme === 'light' ? '#000' : '#fff' }}
      onClick={() => setTheme(theme === 'light' ? 'dark' : 'light')}
    >
      Toggle Theme
    </button>
  );
};
